# IMU_Sensor - Bachelor's thesis project

Project designed in the "Altium Designer" software. The Zalacznik_1_Schemat_elektroniczny.pdf file contains the electrical schematics.

## PCB sections
- Power supply,
- Microcontroller - STM32L4,
- IMU,
- Network module,
- microSD card.
